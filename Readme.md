The main goals of this package are validate, map fields and transform incoming array.

In this package we use schema to work with array. Schema contains nodes tree which allow you to manipulate with 
fields of array.

Examples of usage:

examples/make_schema_from_array.php - show how to make schema from array and add validation and transformation </br>
examples/schema_version.php - show how to make schema version (json format). You can save it and use to create schema </br>
examples/build_schema.php - show how to build schema manually </br>
